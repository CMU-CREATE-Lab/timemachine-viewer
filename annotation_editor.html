Annotation Markers

Create annotation markers for the above time machine.

Instructions:
Place Marker: Hold SHIFT, Left mouse click.
Move Marker: Hold ALT, Left mouse click a marker, Drag.
Select Marker: Hold ALT, Left mouse click a marker OR Click an entry from the list below.
Run Event: Left click a marker.

To load annotations, paste the code into the text box below and then click the Load button.
To save these annotations, copy the code contained in the text box below and paste it into a new file in your favorite text editor.
